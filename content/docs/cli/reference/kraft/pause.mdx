Pause one or more running unikernels


```
kraft pause [FLAGS] MACHINE [MACHINE [...]]
```

### Examples

```
# Pause a running unikernel
$ kraft pause my-machine

```

## Options

```
      --all           Pause all machines
  -h, --help          help for pause
  -p, --plat string   Set the platform virtual machine monitor driver.  Set to 'auto' to detect the guest's platform and 'host' to use the host platform. (default "auto")
```
