List machine networks.


```
kraft net list [FLAGS]
```

### Examples

```
# List all machine networks
$ kraft network list

# List all machine networks in JSON format
$ kraft network list -o json

# List all machine networks with all information
$ kraft network list -l

```

## Options

```
  -h, --help            help for list
  -l, --long            Show more information
  -o, --output string   Set output format. Options: table,yaml,json,full (default "table")
```

## Options inherited from parent commands

```
  -d, --driver string   Set the network driver. (default "bridge")
```
