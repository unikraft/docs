Remove a network.

```
kraft net remove
```

### Examples

```
# Remove a network
$ kraft network remove my-network

```

## Options

```
  -f, --force   Force removal of the network
  -h, --help    help for remove
```

## Options inherited from parent commands

```
  -d, --driver string   Set the network driver. (default "bridge")
```
