List running unikernels

```
kraft ps [FLAGS]
```

### Examples

```
# List all running unikernels
$ kraft ps

# List all unikernels
$ kraft ps --all

# List all unikernels with more information
$ kraft ps --long

```

## Options

```
  -a, --all             Show all machines (default shows just running)
  -m, --arch string     Filter the list by architecture
  -h, --help            help for ps
  -l, --long            Show more information
  -o, --output string   Set output format. Options: table,yaml,json,list (default "table")
  -p, --plat string     Set the platform virtual machine monitor driver. (default "all")
  -q, --quiet           Only display machine IDs
```
