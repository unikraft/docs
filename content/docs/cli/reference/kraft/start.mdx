Start one or more machines

```
kraft start [FLAGS] MACHINE [MACHINE [...]]
```

### Examples

```
# Start a machine
$ kraft start my-machine

```

## Options

```
      --all           Start all machines
  -d, --detach        Run in background
  -h, --help          help for start
      --no-prefix     When starting multiple machines, do not prefix each log line with the name
  -p, --plat string   Set the platform virtual machine monitor driver.  Set to 'auto' to detect the guest's platform and 'host' to use the host platform. (default "auto")
      --rm            Automatically remove the unikernel when it shutsdown
```
