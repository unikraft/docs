---
title: "kraft compose"
description: Build and run compose projects with Unikraft
---

Build and run compose projects with Unikraft.


```
kraft compose [FLAGS] [SUBCOMMAND|DIR]
```

### Examples

```
# Start a compose project
$ kraft compose up

```

## Options

```
      --env-file string   Set the environment file.
  -f, --file string       Set the Compose file.
  -h, --help              help for compose
```
