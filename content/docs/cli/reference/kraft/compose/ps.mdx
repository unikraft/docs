List running services of current project.

```
kraft compose ps [FLAGS]
```

### Examples

```
# List running services of current project
$ kraft compose ps

```

## Options

```
  -a, --all             Show all machines (default shows just running)
  -h, --help            help for ps
  -l, --long            Show more information
      --orphans         Include orphaned services (default: true)
  -o, --output string   Set output format. Options: table,yaml,json,list (default "table")
  -q, --quiet           Only display machine IDs
```

