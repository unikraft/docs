Stop and remove a compose project.

```
kraft compose down [FLAGS]
```

### Examples

```
# Stop and remove a compose project
$ kraft compose down

```

## Options

```
  -h, --help             help for down
      --remove-orphans   Remove machines for services not defined in the Compose file.
```

## Options inherited from parent commands

```
      --env-file string   Set the environment file.
  -f, --file string       Set the Compose file.
```
