---
title: "kraft vol create"
description: Create a machine volume
---

```
kraft vol create VOLUME
```

### Examples

```
# Create a volume with a randomly generated name
$ kraft volume create

# Create a volume with a specific name
$ kraft volume create my-volume

```

## Options

```
  -h, --help   help for create
```

## Options inherited from parent commands

```
  -d, --driver string   Set the volume driver. (default "9pfs")
```
