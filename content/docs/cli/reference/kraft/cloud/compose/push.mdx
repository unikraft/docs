---
title: "kraft cloud compose push"
description: Push the images services to Unikraft Cloud from a Compose project
---

```
kraft cloud compose push [FLAGS] [COMPONENT]
```

### Examples

```
# Push the nginx service image to Unikraft Cloud
$ kraft cloud compose push nginx

```

## Options

```
  -h, --help   help for push
```
