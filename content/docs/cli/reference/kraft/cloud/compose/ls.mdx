---
title: "kraft cloud compose ls"
description: List service deployments at a given path
---

```
kraft cloud compose ls [FLAGS] [PATH]
```

### Examples

```
# List service deployments at a given path.
$ kraft cloud compose ls /path/to/deployment

# List service deployments in the current directory.
$ kraft cloud compose ls

```

## Options

```
  -h, --help            help for ls
  -o, --output string   Set output format. Options: table,yaml,json,list,raw (default "table")
```
