---
title: "kraft cloud quotas"

```
kraft cloud quotas
```

### Examples

```
# View your resource quota on Unikraft Cloud
$ kraft cloud quota

# View your resource quota on Unikraft Cloud in JSON format
$ kraft cloud quota -o json

```
